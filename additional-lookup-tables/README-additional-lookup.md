
# README-additional-lookup.md

These lookup tables are for fields that are not returned by Geosupport, but could be looked up by fields that are returned.

## files

* `NTA-names-lookup-table.csv`
    * NTA name: Geosupport returns an NTA code, with which you can look up the NTA name.
* `UHF_names.csv`
    * A geography used by DOHMH made of ZCTAs, so it can be looked up by ZCTA.
    * This file let's you look up the UHF name by UHF code.
* `ZIP_ZCTA_MODZCTA_UHF_2020.csv`
    * This file lets you looks up UHF code by ZCTA.
