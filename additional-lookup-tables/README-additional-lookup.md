
# README-additional-lookup.md

These lookup tables are for fields that are not returned by Geosupport, but could be looked up by fields that are returned:
* UHF: a geography used by DOHMH made of ZCTAs, so it can be looked up by ZCTA.
* NTA name: Geosupport returns an NTA code, with wich you can look up the NTA name.

Currently (2026-03-11) this lookup is not available in Geoclient. But the plan is to add it.
